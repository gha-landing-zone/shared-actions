# Shared Actions

A repository to house all shared composite Actions and reusable workflows. 

## Supported Build Languages

- Elixir
- Java
- Node.js
- Python
- Ruby
- Rust
- Scala

## Utility Actions

- Downstream Action Triggering
- Security Scanning
- Service Deploy Notifier
- Tag Enforcement

## Getting Started

```zsh
git clone git@github.com:gha-landing-zone/shared-actions.git
```
